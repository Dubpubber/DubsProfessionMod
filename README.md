# DubsProfessionMod# Project Purpose
DubProfessionMod or DPM is a mod for the Java game "Project Zomboid." In Project Zomboid, the player can choose their profession from a short list of base professions. DPM adds even more professions that fundamentally change the gameplay for the user!
